CSS selectors

Selectors Description Example
Element Selectors Selects HTML elements by their name. p selects all <p> elements.
Class Selectors Selects elements with a specific class attributes. .classname selects elements with 'class="classman"'.
Universal Selectors Selects all elements on the page. '*' selects all elements.
Descendant Selectors Selects an element that is a Descendant of another specified element 'div p' selects all '<p>' elements that are Descendants of '<div>' elements.
Child Selectors Selects elements that are direct children of a specific element. 'ul > li' selects all <li> elements that are direct children of <ul> elements.
Adjacent Sibling selectors Selects an element that is directly Adjacent to another element. 'h2 + p' selects the '<p>' element that is directly Adjacent to <h2>.
Attribute selectors Selects elements based on their attributes values 'input[type="text"]'selects all '<input>' elements with 'type="text"'.
Pseudo-Classes Selects elements based on their state or position ':hover' selects an element when the user hovers over it, ':first-child' selects the first child of its parent, and ':nth-child()' selects elements based on their position within a parent.
Pseudo-Elements Selects and styles a part of an element. '::before' and '::after' create Pseudo-Elements before and after an element's content.
